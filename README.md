# CIFAR-10 Image Classification with PyTorch

This project uses PyTorch to train a Convolutional Neural Network, or CNN, to classify images from the CIFAR-10 dataset.

CIFAR-10 contains 60,000 color images in 10 classes:
airplane, automobile, bird, cat, deer, dog, frog, horse, ship, and truck.

## Purpose

The purpose of this project is to learn how PyTorch can be used for image classification. The project demonstrates how to load a dataset, build a neural network, train the model, test its accuracy, and make predictions.

## Important Concepts

- Tensor: A multidimensional array used by PyTorch.
- Dataset: A collection of examples used for training and testing.
- DataLoader: Splits the dataset into smaller batches.
- CNN: A neural network designed for image data.
- Convolution: A process that finds patterns in an image.
- ReLU: An activation function that helps the model learn non-linear patterns.
- Pooling: Reduces image size while keeping important features.
- Loss Function: Measures how wrong the model is.
- Optimizer: Updates the model weights to improve accuracy.
- Epoch: One full pass through the training dataset.
